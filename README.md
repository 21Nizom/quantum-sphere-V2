# 🌐 Quantum Core Sphere v2.0 ⚡

Футуристичный интерактивный веб-интерфейс «Квантового ядра», отображающий динамическую трехмерную сферу из связанных между собой информационных точек. Проект симулирует работу высокотехнологичной Sci-Fi системы.

## 🌐 Живое демо (Live Demo)
👉 **[Запустить проект в браузере](https://21nizom.github.io/quantum-sphere-V2)**

## 🚀 Ключевые фишки (Features)
- **Интерактивное ядро:** Сложная 3D-сфера, состоящая из динамически связанных вершин (квантовых точек) и линий, создающих эффект живого цифрового организма.
- **Смена цветовых режимов (Shift Color):** При клике по экрану система меняет цвет интерфейса и ядра (доступны зеленый, бирюзовый и критический красный режимы — `SHFT_COLOR: ON_CLICK`).
- **Sci-Fi HUD Интерфейс:** На экране отображаются стилизованные индикаторы состояния системы (`SYSTEM: ACTIVE // CORE_STABLE`).
- **Адаптивность:** Проект полностью оптимизирован под мобильные экраны и плавно работает в любом браузере.

## 🛠️ Технологический стек (Tech Stack)
- HTML5 / CSS3 (Кастомные шрифты и футуристичный дизайн Sci-Fi интерфейса)
- JavaScript (Логика переключения состояний ядра и анимация)
- **Three.js / WebGL Canvas** (Генерация, связь и рендеринг частиц в реальном времени)
